Webová prezentace projektu – Stylové Oblečení

Stručný popis projektu

Webová aplikace Stylové Oblečení představuje jednoduchý e-shop zaměřený na moderní a pohodlnou módu. Projekt slouží jako ukázka propojení frontendové části (HTML, CSS) s backendem v Pythonu a databází, která uchovává informace o produktech, zákaznících a objednávkách.

Stránky jsou vytvořeny v HTML5 a stylovány pomocí CSS3. Web je responzivní a správně se zobrazuje jak na počítači, tak na mobilních zařízeních.

Funkce programu

  • Zobrazení nabídky produktů – hlavní stránka zobrazuje produkty (název, popis, cena, obrázek).
  • Navigace mezi stránkami – menu obsahuje odkazy na Domů, Obchod, Kontakt a Maturita.
  • Kontakt a informace o firmě – stránka kontakt.html obsahuje e-mail, adresu a otevírací dobu.
  • Propojení s databází – Python (Flask/Django) obsluhuje dotazy a načítá data z databáze (např. SQLite/MySQL).

Ukázky použití

  • Uživatel otevře index.html a prohlédne si nabídku produktů.
  • Na stránce Obchod vidí kompletní seznam produktů z databáze.
  • Po kliknutí na „Přidat do košíku“ se data uloží do databáze pomocí Pythonu.
  • Na stránce Kontakt najde zákazník informace o firmě.
  • Stránka Maturita obsahuje popis a dokumentaci projektu.

Použité technologie

  • Frontend: HTML5, CSS3 (responzivní design, CSS proměnné)
  • Backend: Python (Flask nebo Django)
  • Databáze: SQLite / MySQL – tabulky pro produkty, uživatele a objednávky
  • Propojení: Python komunikuje s databází přes API endpointy (např. /products, /cart)

Shrnutí

Projekt Stylové Oblečení demonstruje tvorbu přehledné a moderní webové aplikace pro prezentaci a správu produktů. Cílem je ukázat propojení webového rozhraní s databází a backendem v Pythonu, podobně jako fungují reálné e-shopy.
