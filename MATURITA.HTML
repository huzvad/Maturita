Webová prezentace projektu – Stylové Oblečení

Stručný popis projektu

Webová aplikace Stylové Oblečení představuje jednoduchý e-shop zaměřený na moderní a pohodlnou módu. Projekt slouží jako ukázka propojení frontendové části (HTML, CSS) s backendem v Pythonu a databází, která uchovává informace o produktech, zákaznících a objednávkách.

Stránky jsou vytvořeny v HTML a stylovány pomocí CSS. Web je responzivní a správně se zobrazuje jak na počítači, tak na mobilních zařízeních.

Funkce programu

  • Zobrazení nabídky produktů – hlavní stránka zobrazuje produkty (název, popis, cena, obrázek).
  • Navigace mezi stránkami – menu obsahuje odkazy na Domů, Obchod, Kontakt a Maturita.
  • Kontakt a informace o firmě – stránka kontakt.html obsahuje e-mail, adresu a otevírací dobu.
  • Propojení s databází – Python obsluhuje dotazy a načítá data z databáze.

Ukázky použití

  • Uživatel otevře about.html a prohlédne si nabídku nejprodávanějších produktů.
  • Na stránce Obchod vidí kompletní seznam produktů z databáze.
  • Po kliknutí na „Přidat do košíku“ se data uloží do databáze pomocí Pythonu.
  • Na stránce Kontakt najde zákazník informace o firmě.
  • Stránka Maturita obsahuje popis a dokumentaci projektu.

Použité technologie

  • Frontend: HTML5, CSS3 (responzivní design, CSS proměnné)
  • Backend: Python (Flask nebo Django)
  • Databáze: MySQL – tabulky pro produkty, uživatele a objednávky
  • Propojení: Python komunikuje s databází přes API endpointy (např. /products, /cart)

Seznam použitých algoritmů a knihoven

1. Načtení produktů z databáze

Tento algoritmus zajišťuje načtení dat o produktech z databáze pomocí Pythonu a jejich zobrazení na stránce Obchod. Backend provede SQL dotaz, načte všechny produkty a převede výsledek do formátu JSON, který odešle na frontend. 

Funkce nacti_produkty():
    připoj_se_k_databazi()
    vysledek = proveď_SQL_dotaz("SELECT * FROM produkty")
    produkty = převod_na_JSON(vysledek)
    vrať produkty

2. Přidání produktu do košíku

Po kliknutí na tlačítko „Přidat do košíku“ odešle frontend požadavek na server, kde Python uloží vybraný produkt do databáze (nebo do dočasné tabulky košíku). Tento postup umožňuje sledovat, co si uživatel vybral. 

Funkce pridat_do_kosiku(id_produktu, uzivatel):
    připoj_se_k_databazi()
    vloz_do_tabulky("kosik", id_produktu, uzivatel)
    potvrď_změny()

3. Výpočet celkové ceny objednávky

Algoritmus slouží ke spočítání konečné částky v nákupním košíku. Načte všechny položky, jejich ceny a vypočítá součet. 

Funkce spocitej_celkovou_cenu(kosik):
    celkem = 0
    pro každou položku v kosik:
        celkem += položka.cena * položka.mnozstvi
    vrať celkem
